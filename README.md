# Dice_Game_Project

Dice game built to test the knowledge of the tech stack learned

## Tech stack

- Javascript ES6
- CSS3
-HTML5

## Description

- It's a simple Dice game.
- On each refresh random number will be Drawn
- Based on numbers on Dice Players (1 or 2) wins, or draws
